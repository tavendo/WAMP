Protocol Specification
======================

The WAMP specification is authored in Markdown and final specification documents
are generated from the source files in different target formats:

* `WAMP Specification (W3C HTML format) <_static/gen/wamp_latest.html>`_
* `WAMP Specification (IETF HTML format) <_static/gen/wamp_latest_ietf.html>`_
* `WAMP Specification (IETF TXT format) <_static/gen/wamp_latest_ietf.txt>`_

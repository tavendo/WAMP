# WAMP Enhancement Proposals

WAMP Enhancement Proposals (WEPs) are proposals submitted to the WAMP DAO
for governance voting

## WEP Template

Proposals should contain:

* Summary
* Abstract
* Motivation
* Specification
* Benefits
* Drawbacks
* Vote
